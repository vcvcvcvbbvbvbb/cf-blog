The Stack theme is a card-style theme for Hugo, designed for bloggers who want a clean, minimalist, and responsive website. It features a beautiful left sidebar, powerful widgets, and detailed search functionality.

In this project, we are recreating its aesthetic using React and Tailwind CSS.
