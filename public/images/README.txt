## 图片存放说明

### 图片应该放哪里？

把你的图片放在 `public/images/` 文件夹里。

例如：
- `public/images/avatar.jpg`  （头像图片）
- `public/images/banner.png`  （横幅图片）
- `public/images/posts/xxx.jpg` （文章配图）

### 在代码或 Markdown 中怎么引用？

直接使用以 `/` 开头的路径，它会自动指向你的 `public` 文件夹。

**Markdown 中引用图片：**
```markdown
![头像](/images/avatar.jpg)
```
